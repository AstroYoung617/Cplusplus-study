说明下列循环的含义并改正其中的错误。

(a) for (int ix = 0; ix != sz; ++ix) { /* ... */ }
    if (ix != sz)
    	// . . .
(b) int ix;
    for (ix != sz; ++ix) { /* ... */ }
(c) for (int ix = 0; ix != sz; ++ix, ++sz) { /*...*/ }

解答： 
    a) for语句头中定义的变量只在循环体中可见，if语句不能使用。ix应该单独定义在外面。
    b) for语句头缺少初始化，可以添加一个分号。
    c) sz也在不断地增长，条件表达式永远为true，将无限循环。
