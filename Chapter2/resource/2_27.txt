下面的哪些初始化是合法的？请说明原因。

a) int i = -1, &r = 0;				//非法，引用初始化必须绑定对象，不能绑定字面值
b) int *const p2 = &i2;				//合法，p2是一个常量指针，初始化为i2的地址，之后不能改变
c) const int i = -1, &r = 0;		//非法，引用初始化必须绑定对象，不能绑定字面值（错，合法，常量引用可以绑定非常量的字面值和对象）
d) const int *const p3 = &i2;		//合法，p3是一个指向常量对象的常量指针，将i2的地址赋值给它
e) const int *p1 = &i2;				//合法，p1是一个指针，它指向整型常量，将i2的地址赋给它，它不能改变i2的值，但是可以改变自身的地址
f) const int &const r2;				//不合法，引用前面不能加const
g) const int i2 = i, &r = i;		//合法，i2是整型常量初始化为i，常量引用绑定i（无论i是否是常量都可以满足）
